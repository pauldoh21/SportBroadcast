# SportBroadcast

# Pre-requisites

- [just](https://github.com/casey/just/releases/download/1.50.0/just-1.50.0-x86_64-pc-windows-msvc.zip)
- [pnpm](https://pnpm.io/installation#on-windows)

# Examples

- Run the backend: `just backend`

- Run the frontend: `just frontend`

- Run both at the same time: `just all`
